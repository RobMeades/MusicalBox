# Introduction
The files in here are are generated using [Gearotic](https://www.gearotic.com/), a Windows-based software package designed for creating gears.  Mostly I use it to get a good gear shape that I can import into Blender, I tend not to use it for gear simulation etc.

Anyway, the files in here are:

- `rotation_gears.gth`: the Gearotic file that contains all of the gears for the musical box project.
- `rack.stl`: export of the gear rack that was imported into Blender and circularised for the base of the musical box.
- `small.stl`: a pinion gear to match the rack, imported into Blender and curved to match the circularised rack.

For more information on how these files were employed in the project see [here](https://www.meades.org/misc/musical_box/musical_box.html#Rotation).
